02 Offensive / Endpoint

Rogue USB — HID Keystroke Injection Attack

Built a Digispark-based HID attack that simulates a rogue USB device injecting keystrokes to execute automated payloads on a target machine. The goal was to understand physical access threat vectors and test endpoint assumptions — specifically, why organizations need to control removable media and physical port access.
